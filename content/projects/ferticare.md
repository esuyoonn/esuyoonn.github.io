**Samshin (산신) Collaboration** | *Oct 2025 – Feb 2026*

## Overview

Led the end-to-end development of **FertiCare**, a domain-specific chatbot for infertility patients, built in collaboration with fertility startup Samshin. The system uses a multi-agent architecture to handle the complexity of fertility care questions — routing queries, managing context, and providing accurate, empathetic responses.

## Responsibilities

- Sole developer responsible for model architecture design and implementation
- Conducted iterative experimentation to optimize agent routing and response quality
- Participated in client meetings with Samshin to align on product requirements and clinical accuracy

## Architecture

The system employs a multi-agent setup (MAS) where specialized sub-agents handle distinct domains — medical Q&A, emotional support, and treatment process guidance — coordinated by an orchestrator agent.

## Technologies
Gemini API, LangChain, Python
