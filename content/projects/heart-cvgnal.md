## Overview

Built a real-time system to measure interpersonal affinity by analyzing non-verbal cues from facial expressions. The system captures live video, extracts facial landmarks, and uses a vision-language model to interpret emotional and relational context between subjects.

## Approach

- Used **MediaPipe** for efficient, real-time facial landmark detection on live video streams
- Integrated **Claude VLM** to perform contextual interpretation of detected expressions and micro-expressions
- Developed a scoring mechanism to quantify interpersonal affinity signals over time

## Technologies
MediaPipe, Claude API (VLM), Python, HTML
